StyleBay 

A domain-model MVC project for managing dresses, orders, users, and categories.
Includes models, DTOs, repositories, services, and controllers.

Features:

- Domain models for Dress, Order, User, etc.
- DTOs for data transfer
- Repository and Service patterns
- Email configuration example
- MVC controllers
- Migrations included for database setup
